# Gerai Kompas

Prototype website ecommerce responsif untuk Gerai Kompas.

## Fitur

- Header dengan logo, search, wishlist, cart, dan tombol akun.
- Navigasi kategori horizontal.
- Hero visual besar seperti referensi home.
- Popular categories.
- Product grid dengan rating, harga diskon, wishlist, filter kategori, search, dan sort.
- Cart drawer dengan quantity dan total.
- Responsive untuk desktop, iPad/tablet, dan mobile phone.

## Menjalankan

Buka `index.html` langsung di browser, atau jalankan server statis dari folder ini:

```bash
python3 -m http.server 8000
```

Lalu buka `http://localhost:8000`.
